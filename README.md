## 📝 Project Overview
An interactive web-based application designed to help users practice and improve their mental arithmetic skills through gamified learning. The game features adaptive difficulty progression, real-time statistics tracking, and instant feedback to create an engaging learning experience. 

## ✨ Key Features

### Two Game Modes:
* Fixed Difficulty: Choose your preferred difficulty level
* Level Progression: Automatically scales from beginner to expert (10 levels)
* Multiple Operations: Addition, Subtraction, Multiplication, Division, and Mixed
* Real-time Statistics: Score, streak, accuracy, and time tracking
* Instant Feedback: Auto-advance on correct answers with visual animations
* Responsive Design: Works seamlessly on desktop and mobile devices
* Optional Backend: Score persistence and leaderboard functionality

## 🛠️ Technologies Used

Frontend: HTML5, CSS3, JavaScript (ES6)
Backend (Optional): Python, Flask
Database (Optional): SQLite
Design: Glassmorphism, CSS Animations, Flexbox/Grid

## 📝 プロジェクト概要
暗算スキルを向上させるためのインタラクティブなWebアプリケーション。ゲーミフィケーションを通じて楽しく学習できます。適応型難易度調整、リアルタイム統計追跡、即座のフィードバックにより、魅力的な学習体験を提供します。

## ✨ 主な機能
- **2つのゲームモード:**
  - 固定難易度：好みの難易度を選択
  - レベル進行：初心者からエキスパートまで自動スケーリング（10レベル）
- **複数の演算子:** 足し算、引き算、掛け算、割り算、混合
- **リアルタイム統計:** スコア、連続正解数、正解率、残り時間
- **即座のフィードバック:** 正解時の自動進行とビジュアルアニメーション
- **レスポンシブデザイン:** デスクトップとモバイルで完璧に動作
- **オプションのバックエンド:** スコア保存とリーダーボード機能

## 🛠️ 使用技術
- **フロントエンド:** HTML5, CSS3, JavaScript (ES6)
- **バックエンド（オプション）:** Python, Flask
- **データベース（オプション）:** SQLite
- **デザイン:** グラスモーフィズム、CSSアニメーション、Flexbox/Grid
